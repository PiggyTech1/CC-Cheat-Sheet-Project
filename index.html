CSS Display and Positioning

CSS Position Functions

Syntax Description
position: relative; An element's position is relative to its default position on the page.
position: absolute; An element's position is relative to its closest positioned parent element. It can be pinned to any part of the web page, but the element will still move with the rest of the document when the page is scrolled.
position: fixed; An element's position can be pinned to any part of the web page. The element will remain in view no matter what.
position: sticky; An element can stick to a defined offset position when the user scrolls its parent container.
z-index: int; Specifies how far back or how far forward an element appears on the page when it overlaps other elements.

CSS Display Functions

Syntax Description
display: inline; Elements take up as little space as possible, and they cannot have manually adjusted width or height.
display: block; Elements take up the width of their container and can have manually adjusted heights.
display: inline-block; Elements can have set width and height, but they can also appear next to each other and do not take up their entire container width.

Additional CSS Display and Positioning Functions

Syntax Description
float: position; Moves elements as far left or as far right as possible on a web page.
clear: position; Clears an element's left or right side (or both + none)
